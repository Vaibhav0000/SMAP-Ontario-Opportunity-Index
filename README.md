<h1 align="center">Senior Mobility and Accessibility Project (SMAP)</h1>

<p align="center">
  <strong>Ontario CMA-Level Opportunity Intelligence Framework</strong>
</p>

<hr/>

<h2>Executive Summary</h2>

<p>Ontario’s senior population is growing rapidly.</p>

<p>
The mobility device ecosystem (wheelchairs, walkers, scooters, lifts, ramps, cushions) 
is not constrained by total supply. It is constrained by geographic misallocation.
</p>

<p>
SMAP integrates Supply, Access, and Demand into a standardized CMA-level Opportunity 
Index (0–100) to identify where service gaps and commercial expansion opportunities are largest.
</p>

<p><strong>The market is not saturated. It is misaligned.</strong></p>

<hr/>

<h2>The Core Problem</h2>

<p>Ontario has approximately:</p>

<ul>
  <li>94,000 licensed mobility devices</li>
  <li>5,806 registered vendors</li>
  <li>2 million seniors aged 65+</li>
</ul>

<p>Per-capita device access varies significantly:</p>

<ul>
  <li>Barrie: 3.3 devices per 10,000 seniors</li>
  <li>Kitchener-Waterloo: 2.5 devices per 10,000 seniors</li>
  <li>Ottawa: 0.3 devices per 10,000 seniors</li>
  <li>Oshawa: 0.3 devices per 10,000 seniors</li>
</ul>

<p><strong>The issue is not the absence of supply.<br/>
It is concentration in the wrong cities.</strong></p>

<h2>Methodology</h2>

<h3>Three-Pillar Framework</h3>

<h4>Supply</h4>
<ul>
  <li><strong>Source:</strong> MDALL (Health Canada)</li>
  <li><strong>Measures:</strong> Licensed manufacturers and active device counts</li>
</ul>

<h4>Access</h4>
<ul>
  <li><strong>Source:</strong> Ontario ADP Vendor Registry</li>
  <li><strong>Measures:</strong> Vendor density per CMA</li>
</ul>

<h4>Demand</h4>
<ul>
  <li><strong>Source:</strong> Statistics Canada Census and CSD</li>
  <li><strong>Measures:</strong> Senior population, disability prevalence, usage rates</li>
</ul>

<h3>Standardization</h3>

<ul>
  <li>Per-capita normalization (per 10,000 seniors)</li>
  <li>Z-score standardization</li>
  <li>Rank scaling to 0–100</li>
  <li>Equal-weight composite scoring</li>
</ul>

<p><strong>Opportunity Score formula:</strong></p>

<pre>
Opportunity Score = (Demand Gap + Access Deficit + Equity Need) / 3
</pre>

<p><strong>Components:</strong></p>

<ul>
  <li><strong>Demand Gap:</strong> Percentage of seniors not using assistive technology</li>
  <li><strong>Access Deficit:</strong> Inverted vendor and device density</li>
  <li><strong>Equity Need:</strong> Composite of visible minority %, Indigenous %, and senior population growth</li>
</ul>

<hr/>

<h2>Key Findings</h2>

<p><strong>Demand is accelerating:</strong></p>

<ul>
  <li>Usage increased from 31.5% to 41.6% in five years</li>
  <li>~114,000 additional senior users added between 2021–2025</li>
</ul>

<p><strong>Larger cities underperform per capita:</strong></p>

<ul>
  <li>Barrie: 3.3</li>
  <li>Kitchener-Waterloo: 2.5</li>
  <li>Hamilton: 1.3</li>
  <li>Ottawa: 0.3</li>
  <li>Oshawa: 0.3</li>
</ul>

<p><strong>Equity overlaps with weak access:</strong></p>

<p>
Highest Equity Need scores include Ottawa, Toronto, Windsor, 
Hamilton, and London.
</p>

<p>Opportunity is primarily access-driven.</p>

<p><strong>Correlation between Access Deficit and Opportunity Score: r ≈ 0.57</strong></p>

<h2>Data Pipeline</h2>

<ul>
  <li>MDALL licence cleaning and deduplication</li>
  <li>ADP vendor normalization</li>
  <li>FSA-to-CMA spatial overlay</li>
  <li>Provincial demand allocation to CMAs</li>
  <li>Per-capita normalization</li>
  <li>Z-score calculation</li>
  <li>Composite scoring</li>
  <li>Power BI dashboard deployment</li>
</ul>

<p>All transformations and assumptions are documented.</p>

<hr/>

<h2>Limitations</h2>

<ul>
  <li>Provincial-level demand allocated proportionally to CMAs</li>
  <li>MDALL licence address used as proxy for device distribution</li>
  <li>Some Statistics Canada suppression required proportional estimation</li>
  <li>Data sources not perfectly temporally aligned</li>
</ul>

<p>All limitations are transparently documented in the full report.</p>
